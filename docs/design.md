# 暗黑 like 自动挂机刷宝游戏 Prototype

## 游戏概述

一款受《暗黑破坏神》启发的自动挂机刷宝游戏。玩家无需手动操作战斗，角色会自动探索地牢、击败怪物、收集装备。玩家的核心乐趣在于装备搭配、词条优化和角色成长。

---

## 核心系统

### 一、角色系统

#### 1.1 基础属性
| 属性 | 说明 |
|------|------|
| 生命值 (HP) | 角色血量，归零则死亡 |
| 攻击力 (ATK) | 基础伤害值 |
| 防御力 (DEF) | 减少受到的伤害 |
| 攻击速度 (AS) | 每秒攻击次数 |
| 暴击率 (CR) | 暴击概率 (0-100%) |
| 暴击伤害 (CD) | 暴击时的伤害倍率 |
| 移动速度 (MS) | 地牢移动效率 |

#### 1.2 等级系统
- 击杀怪物获得经验值
- 升级提升基础属性
- 每级获得 1 点属性点，可自由分配

### 二、背包系统

#### 2.1 背包设计
- **容量**: 40 格 (5 行 x 8 列)
- **物品类型**: 装备、消耗品、材料
- **操作**: 点击查看详情、长按拖拽排序、双击快速装备

#### 2.2 物品稀有度
| 稀有度 | 颜色 | 掉落权重 | 词条数量 |
|--------|------|----------|----------|
| 普通 (Common) | 灰色 | 60% | 0-1 |
| 魔法 (Magic) | 蓝色 | 25% | 1-2 |
| 稀有 (Rare) | 黄色 | 12% | 2-4 |
| 传奇 (Legendary) | 橙色 | 2.5% | 3-5 |
| 神话 (Mythic) | 红色 | 0.5% | 4-6 + 特殊效果 |

#### 2.3 物品筛选
- 按稀有度筛选
- 按装备部位筛选
- 一键出售白装/蓝装
- 一键分解功能

### 四、装备词条系统

#### 4.1 词条类型

**攻击类词条**
| 词条名 | 效果范围 | 权重 |
|--------|----------|------|
| 力量 | ATK +5-50 | 常见 |
| 狂暴 | ATK +10-30% | 稀有 |
| 暴击 | CR +2-15% | 稀有 |
| 暴击伤害 | CD +10-80% | 稀有 |
| 攻速 | AS +5-25% | 较稀有 |
| 穿透 | 无视 DEF +10-50% | 极稀有 |
| 吸血 | 攻击回血 1-10% | 极稀有 |

**防御类词条**
| 词条名 | 效果范围 | 权重 |
|--------|----------|------|
| 体质 | HP +20-200 | 常见 |
| 坚韧 | DEF +5-40 | 常见 |
| 生命回复 | 每秒回复 1-20 HP | 较稀有 |
| 伤害减免 | 受到伤害 -2-15% | 稀有 |
| 闪避 | 闪避率 +2-12% | 稀有 |

**特殊词条 (仅传奇/神话)**
| 词条名 | 效果 |
|--------|------|
| 连击 | 每次攻击有 20% 概率额外攻击一次 |
| 旋风斩 | 攻击时 15% 概率对周围敌人造成 AOE 伤害 |
| 复活甲 | 死亡时有 30% 概率复活，恢复 50% HP |
| 掠夺者 | 击杀怪物时 10% 概率额外掉落一件装备 |
| 狂战士 | 血量低于 30% 时，ATK +100% |
| 不朽 | 受到的致命伤害转为 1 HP (120秒冷却) |

#### 4.2 词条生成规则
1. 装备掉落时根据稀有度随机生成词条数量
2. 词条类型按权重随机选取
3. 词条数值在范围内随机
4. 同类型词条不重复
5. 高稀有度装备可能出现更高数值范围的词条

#### 4.3 词条强化
- 使用材料可重铸装备词条
- 消耗同稀有度装备可提升词条数值

### 五、地牢自动探索系统

#### 5.1 地牢结构
```
地表村庄 (安全区)
    ↓
第 1-5 层: 荒芜矿坑 (怪物等级 1-10)
    ↓
第 6-10 层: 幽暗森林 (怪物等级 11-20)
    ↓
第 11-15 层: 熔岩深渊 (怪物等级 21-35)
    ↓
第 16-20 层: 深渊领域 (怪物等级 36-50)
    ↓
无尽深渊 (无限层数，难度递增)
```

#### 5.2 自动战斗逻辑
```
优先级 1: 检测生命值，低于阈值使用药水或撤退
优先级 2: 主动攻击最近的敌人
优先级 3: 拾取掉落物品
优先级 4: 向未探索区域移动
优先级 5: 寻找下一层入口
```

#### 5.3 怪物系统

**怪物类型**
| 类型 | 特点 |
|------|------|
| 普通 | 标准属性，无特殊技能 |
| 精英 | 150% 血量，1-2 个技能 |
| 稀有 | 200% 血量，2-3 个技能，必掉装备 |
| Boss | 500% 血量，独特技能，必掉稀有+装备 |

**怪物属性公式**
```
怪物 HP = 基础值 × (1 + 层数 × 0.15)
怪物 ATK = 基础值 × (1 + 层数 × 0.12)
经验值 = 基础值 × (1 + 层数 × 0.1)
```

#### 5.4 掉落系统
- 普通怪物: 30% 掉落装备，70% 掉落金币/材料
- 精英怪物: 100% 掉落装备 (至少蓝装)
- 稀有怪物: 100% 掉落装备 (至少稀有)
- Boss: 必掉 3-5 件装备 (至少 1 件传奇)

#### 5.5 探索状态
角色始终处于以下状态之一：
- **探索中**: 在当前层移动寻找怪物
- **战斗中**: 与怪物交战
- **搜刮中**: 拾取战利品
- **休息中**: 生命值低，自动回血
- **转移中**: 前往下一层

### 八、挂机奖励系统

#### 8.1 离线收益
- 离线时角色自动战斗
- 离线收益 = 在线收益 × 50%
- 最长计算 24 小时离线收益

#### 8.2 每日奖励
- 每日首次登录: 金币 × 当前层数 × 100
- 连续登录奖励递增
- 每日任务系统
