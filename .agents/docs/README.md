# .agents/docs 使用说明

该目录用于功能开发文档，按功能分类归档。

## 目录分类

- `00-templates`: 文档模板
- `01-gameplay-loop`: 核心玩法循环（回合、流程、状态机）
- `02-combat`: 战斗相关（伤害、技能、站位、目标选择）
- `03-economy`: 经济系统（金币、商店、刷新、利息）
- `04-progression`: 成长系统（等级、羁绊、升级、解锁）
- `05-ui-ux`: 界面与交互（HUD、菜单、反馈）
- `06-content`: 内容生产（棋子、道具、关卡、配置）
- `07-engine-tech`: 引擎与技术（性能、资源加载、架构、工具链）
- `08-bugfix`: 缺陷修复方案与复盘
- `99-archive`: 已废弃或历史方案归档

## 开发流程（强制）

1. 新增功能点前，先在对应分类目录创建开发文档。
2. 文档命名：`YYYY-MM-DD-功能短名.md`（例如：`2026-04-15-shop-reroll.md`）。
3. 文档通过评审后再开始编码。
4. 功能开发过程中持续更新文档状态与决策记录。
5. 功能完成后，将结果同步到 `FEATURES.md`。
