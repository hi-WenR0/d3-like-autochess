## 2. 目标与范围

- 目标：让背包手动拆解支持传说与神话装备，补齐高稀有度装备的拆解入口。
- 本次范围（In Scope）：
  - 在背包手动拆解区域增加“拆<=传说”和“拆<=神话”按钮
  - 保持现有统一拆解结算逻辑
  - 同步更新文档与跟踪状态
- 非本次范围（Out of Scope）：
  - 装备锁定保护
  - 拆解确认弹窗
  - 高稀有度差异化拆解产物

## 3. 需求与验收标准

- 功能需求：
  - 玩家可在背包面板中手动拆解传说装备。
  - 玩家可在背包面板中手动拆解神话装备。
  - 高稀有度拆解继续沿用现有精华产出规则。
- 验收标准（可检查）：
  - [x] 背包面板显示“拆<=传说”按钮
  - [x] 背包面板显示“拆<=神话”按钮
  - [x] 点击后正确移除对应装备并累计精华
  - [x] 地牢中仍然禁止手动拆解
