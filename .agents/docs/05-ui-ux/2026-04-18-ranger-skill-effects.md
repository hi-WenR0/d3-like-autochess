# 功能开发文档：游侠主动技能特效接入

## 1. 基本信息

- 日期：2026-04-18
- 负责人：Codex
- 分类：05-ui-ux
- 关联功能：游侠主动技能美术特效接入
- 状态：Testing

## 2. 目标与范围

- 目标：
  - 将 `public/assets/youxia/skills` 中现有的 `穿风箭` 与 `爆裂箭雨` 技能特效接入游戏内播放。
  - 保持现有技能数值、命中与自动施法逻辑不变，只补足视觉表现。
- 本次范围（In Scope）：
  - 游侠基础主动 `穿风箭`
  - 游侠转职后通用主动 `爆裂箭雨`
  - 扩展玩家技能特效加载系统，支持按职业切换资源根目录，以及支持单图逐帧和横向 spritesheet 两种素材形态
- 非本次范围（Out of Scope）：
  - 游侠其它专精选项技能特效
  - 被动技能与触发技能特效
  - 对原始素材图片做离线重绘或重新导出

## 3. 需求与验收标准

- 功能需求：
  1. `穿风箭` 释放时播放长条箭矢特效，方向对准目标。
  2. `穿风箭` 的第一帧顶部应从玩家位置出发，形成“射箭”观感。
  3. `爆裂箭雨` 在目标脚下播放范围特效。
  4. `爆裂箭雨` 支持从单张横向 `spritesheet.png` 切帧播放。
- 验收标准：
  - [ ] 游侠进入战斗后可看到 `穿风箭` 特效从玩家位置朝目标方向发出
  - [ ] `爆裂箭雨` 在目标脚下播放，不落在玩家脚下
  - [ ] 两个技能特效均可正常播放完毕并自动销毁
  - [ ] `npx tsc --noEmit` 与 `npm run build-nolog` 通过

## 4. 方案设计

- 玩法/交互设计：
  - `穿风箭` 使用玩家原点生成，设置顶部锚点与固定角度补偿，使长图顶端贴住玩家并朝目标延展。
  - `爆裂箭雨` 继续沿用目标中心 AOE 落点，在目标脚下直接播放。
- 技术设计：
  - 扩展 `player-skill-visuals.ts`，支持按职业映射资源根目录。
  - 在技能特效配置中增加素材类型、锚点与 spritesheet 帧尺寸配置。
  - `Preloader` 对逐帧 PNG 与 spritesheet 分别加载。
  - `Game` 中的技能动画注册逻辑根据素材类型分别创建动画。

## 5. 实现计划

- 新增游侠两个技能特效配置。
- 扩展加载与动画创建逻辑，兼容 `frames` / `spritesheet`。
- 为 `穿风箭` 增加顶部锚点与固定旋转补偿。
- 为 `爆裂箭雨` 配置目标脚下播放。
- 记录功能追踪并完成构建验证。

## 6. 测试计划

- 手工测试步骤：
  1. 使用游侠进入战斗，观察 `穿风箭` 是否从玩家位置朝敌人发出。
  2. 转职后触发 `爆裂箭雨`，确认特效落在目标脚下。
  3. 多次连续释放两技能，确认不会残留 sprite 或报错。
- 边界情况：
  - 目标位于玩家上、下、左、右与斜向时，`穿风箭` 朝向均应正确。
  - `爆裂箭雨` 触发时若素材不存在，应被安全跳过而不是卡死。
