# 功能开发文档：技能特效优化

## 1. 基本信息

- 日期：2026-04-17
- 负责人：AI Agent
- 分类：05-ui-ux
- 关联功能：视觉效果系统、怪物技能系统
- 状态：Completed

## 2. 目标与范围

- 目标：
  - 优化所有技能特效，使其更加精细、合理
  - 调整粒子数量、大小、持续时间等参数
  - 提升视觉体验的同时保持性能稳定

- 本次范围（In Scope）：
  - 粒子效果参数调整（数量、大小、速度、生命周期）
  - 技能预警圈视觉优化
  - 死亡粒子效果缩放调整
  - 受击闪光效果强度调整
  - 可能添加特效开关设置

- 非本次范围（Out of Scope）：
  - 新增特效类型
  - 完全重写特效系统
  - 添加高级图形设置面板

## 3. 需求与验收标准

- 功能需求：
  1. 粒子数量优化
     - 普通怪死亡：20 → 12
     - 精英怪死亡：30 → 18
     - 稀有怪死亡：40 → 24
     - Boss死亡：60 → 35
     - 受击效果：8 → 5

  2. 粒子大小调整
     - 整体缩小约30%，减少视觉冲击

  3. 技能预警圈优化
     - 降低初始透明度
     - 缩短预警时间（部分技能）
     - 调整颜色饱和度

  4. 受击闪光效果
     - 缩短持续时间（100ms → 60ms）

- 验收标准（可检查）：
  - [x] 粒子效果不再显得过于夸张
  - [x] 特效不影响游戏性能（60fps稳定）
  - [x] 视觉反馈仍然清晰可辨
  - [x] Boss战特效密度合理

## 5. 实现计划

- 任务拆分：

  1. 调整 `particle-configs.ts` 中所有粒子配置参数
  2. 修改 Game.ts 中受击闪光持续时间
  3. 调整技能预警圆环视觉效果
  4. 测试各稀有度怪物的视觉效果
  5. 验证性能（确保帧率稳定）

- 预计改动文件：
  - 修改: `src/game/configs/particle-configs.ts`
  - 修改: `src/game/scenes/Game.ts`

- 风险点与回滚方案：
  - 风险1：特效过于微弱，失去视觉反馈意义
    - 缓解：保持基本可辨识度，仅降低强度
    - 回滚：恢复原配置值

  - 风险2：玩家对特效强度偏好不同
    - 缓解：考虑添加"高/中/低"特效质量设置
    - 回滚：提供多种预设配置

## 6. 测试计划

- 手工测试步骤：

  1. 进入地牢，攻击普通怪，观察受击闪光效果
  2. 击杀普通怪，观察死亡粒子效果
  3. 找到精英怪，测试精英特效
  4. 找到Boss，测试Boss技能预警和死亡特效
  5. 在同屏多怪情况下测试性能

- 边界情况：
  - 快速击杀多只怪物时粒子不堆积
  - Boss战特效不会过度遮挡视线
  - 低端设备上帧率保持稳定

- 性能影响检查：
  - 粒子数量上限检查
  - 内存使用无明显增长
  - 60fps稳定运行
